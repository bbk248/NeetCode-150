House Robber II

Problem: You are a professional robber planning to rob houses along a street. Each house has a certain amount of money stashed. All houses at this place are arranged in a circle. That means the first house is the neighbor of the last one. Meanwhile, adjacent houses have security systems connected, and it will automatically contact the police if two adjacent houses were broken into on the same night.

Given an integer array nums representing the amount of money of each house, return the maximum amount of money you can rob tonight without alerting the police.

Example 1:
Input: nums = [2,3,2]
Output: 3
Explanation: You cannot rob house 1 (money = 2) and then rob house 3 (money = 2), because they are adjacent houses.
Example 2:
Input: nums = [1,2,3,1]
Output: 4
Explanation: Rob house 1 (money = 1) and then rob house 3 (money = 3).
Total amount you can rob = 1 + 3 = 4.
Example 3:
Input: nums = [1,2,3]
Output: 3
Constraints:

üí° Approach

Circular constraint: can't rob both first and last. Solve for [0...n-2] and [1...n-1], return max.

üî• Optimal Solution

class Solution {
    public int rob(int[] nums) {
        int n = nums.length;
        
        // Edge case: only one house
        if (n == 1) {
            return nums[0];
        }
        
        // Edge case: only two houses
        if (n == 2) {
            return Math.max(nums[0], nums[1]);
        }
        
        // Case 1: Rob houses from 0 to n-2 (exclude last house)
        int robFirst = robLinear(nums, 0, n - 2);
        
        // Case 2: Rob houses from 1 to n-1 (exclude first house)
        int robLast = robLinear(nums, 1, n - 1);
        
        // Return maximum of both cases
        return Math.max(robFirst, robLast);
    }
    
    private int robLinear(int[] nums, int start, int end) {
        int prev2 = 0;
        int prev1 = 0;
        
        for (int i = start; i <= end; i++) {
            int current = Math.max(prev1, prev2 + nums[i]);
            prev2 = prev1;
            prev1 = current;
        }
        
        return prev1;
    }
}

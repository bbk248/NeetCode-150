üìã Problem Statement

Burst Balloons

Problem: You are given n balloons, indexed from 0 to n - 1. Each balloon is painted with a number on it represented by an array nums. You are asked to burst all the balloons.

If you burst the ith balloon, you will get nums[i - 1] * nums[i] * nums[i + 1] coins. If i - 1 or i + 1 goes out of bounds of the array, then treat it as if there is a balloon with a 1 painted on it.

Return the maximum coins you can collect by bursting the balloons wisely.

Example 1:
Input: nums = [3,1,5,8]
Output: 167
Explanation:
nums = [3,1,5,8] --> [3,5,8] --> [3,8] --> [8] --> []
coins = 3*1*5 + 3*5*8 + 1*3*8 + 1*8*1 = 15 + 120 + 24 + 8 = 167
Example 2:
Input: nums = [1,5]
Output: 10
Constraints:

üí° Approach

Interval DP: Think backwards - which balloon to burst LAST in each subrange. Add boundary balloons with value 1.

üî• Optimal Solution

class Solution {
    public int maxCoins(int[] nums) {
        int n = nums.length;
        
        // Add boundary balloons with value 1
        int[] balloons = new int[n + 2];
        balloons[0] = 1;
        balloons[n + 1] = 1;
        for (int i = 0; i < n; i++) {
            balloons[i + 1] = nums[i];
        }
        
        // dp[i][j] = max coins from bursting balloons between i and j (exclusive)
        int[][] dp = new int[n + 2][n + 2];
        
        // Length of the interval
        for (int len = 2; len <= n + 1; len++) {
            // Start of the interval
            for (int i = 0; i <= n + 1 - len; i++) {
                int j = i + len;
                
                // Try each balloon k as the last one to burst in range (i, j)
                for (int k = i + 1; k < j; k++) {
                    // Coins from bursting k last = balloons[i] * balloons[k] * balloons[j]
                    // Plus coins from left and right subproblems
                    int coins = balloons[i] * balloons[k] * balloons[j] + dp[i][k] + dp[k][j];
                    dp[i][j] = Math.max(dp[i][j], coins);
                }
            }
        }
        
        return dp[0][n + 1];
    }
}

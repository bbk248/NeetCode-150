üí° Approach

Dynamic Programming: For number i, count = count of (i >> 1) + (i & 1). Use previously computed results.

üî• Optimal Solution

class Solution {
    public int[] countBits(int n) {
        int[] dp = new int[n + 1];
        
        for (int i = 1; i <= n; i++) {
            // dp[i] = dp[i >> 1] + (i & 1)
            // i >> 1 is i divided by 2 (removing rightmost bit)
            // i & 1 is 1 if i is odd, 0 if i is even (rightmost bit)
            dp[i] = dp[i >> 1] + (i & 1);
        }
        
        return dp;
    }
}

Time: O(n) - single pass through all numbers

Space: O(1) - excluding the output array

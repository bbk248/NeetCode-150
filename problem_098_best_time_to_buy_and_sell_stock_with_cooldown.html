üìã Problem Statement

Best Time to Buy and Sell Stock with Cooldown

Problem: You are given an array prices where prices[i] is the price of a given stock on the ith day.

Find the maximum profit you can achieve. You may complete as many transactions as you like (i.e., buy one and sell one share of the stock multiple times) with the following restrictions:

  • After you sell your stock, you cannot buy stock on the next day (i.e., cooldown one day).

Note: You may not engage in multiple transactions simultaneously (i.e., you must sell the stock before you buy again).

Example 1:
Input: prices = [1,2,3,0,2]
Output: 3
Explanation: transactions = [buy, sell, cooldown, buy, sell]
Example 2:
Input: prices = [1]
Output: 0
Constraints:

üí° Approach

State machine DP: Track three states - holding stock, sold (cooldown), and resting. Handle transitions between states.

üî• Optimal Solution

class Solution {
    public int maxProfit(int[] prices) {
        if (prices.length <= 1) {
            return 0;
        }
        
        // State definitions:
        // hold: maximum profit when holding a stock
        // sold: maximum profit when just sold (cooldown day)
        // rest: maximum profit when resting (can buy)
        
        int hold = -prices[0]; // Buy on first day
        int sold = 0;          // Can't sell on first day
        int rest = 0;          // Rest on first day
        
        for (int i = 1; i < prices.length; i++) {
            int prevHold = hold;
            int prevSold = sold;
            int prevRest = rest;
            
            // To hold: either keep holding or buy today (from rest state)
            hold = Math.max(prevHold, prevRest - prices[i]);
            
            // To sell: sell the stock we were holding
            sold = prevHold + prices[i];
            
            // To rest: either keep resting or cooldown after selling
            rest = Math.max(prevRest, prevSold);
        }
        
        // Return max of sold or rest (we don't want to end holding stock)
        return Math.max(sold, rest);
    }
}
